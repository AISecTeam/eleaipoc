漏洞详情

漏洞描述

Ollama 是一个开源的本地机器学习平台，提供了一个简单易用的 API 接口，允许用户创建、训练和使用模型。Ollama 的 API 接口存在安全漏洞，攻击者可通过未授权访问 API 接口，实现删除模型、信息窃取等操作，导致系统安全风险。

本次扫描关键特征

检测到CNVD-2025-04094漏洞，具体特征为：127.0.0.1:11434 的 Ollama 服务存在API可连接,攻击者可未授权访问 Ollama 服务的 API，实现删除模型、信息窃取等操作

安全建议

  • 1. 立即启用 Ollama 服务的认证机制，限制只有授权用户可以访问 API
  • 2. 确保 Ollama 服务只在可信的网络环境中运行，避免暴露于公共网络
  • 3. 定期更新 Ollama 版本，应用最新的安全补丁
  • 4. 对敏感操作（如删除模型）添加额外的权限验证

参考链接

https://www.cnvd.org.cn/flaw/show/CNVD-2025-04094

漏洞描述

Ray 是一个开源的分布式机器学习框架，提供了一个简单易用的 API 接口，允许用户创建、训练和使用模型。Ray 的 API 接口存在远程命令执行漏洞，攻击者可利用该漏洞远程控制Ray服务。

本次扫描关键特征

检测到CVE-2024-57000及CVE2023-48022，具体特征为：创建job(检测用例)成功，获得job_id为raysubmit_7dF9iYZ7sjddXkBX。

安全建议

  • 2. 对 Ray 服务增加访问认证机制，限制未授权访问
  • 3. 避免将 Ray 服务暴露在公共网络
  • 4. 定期更新 Ray 版本，应用最新的安全补丁
  • 5. 对敏感操作（如删除模型）添加额外的权限验证
